GenAI LangGraph Resume Analyzer Agent

Overview

A Generative AI Resume Analyzer Agent built using LangGraph, Gemini, and Python.

The agent reads a resume PDF, extracts technical skills, identifies missing skills compared to a target role, generates a personalized learning roadmap, and supports human approval and email workflows.

Features

* Resume PDF Parsing
* Skill Extraction using Gemini
* Skill Gap Analysis
* Learning Roadmap Generation
* LangGraph State Management
* Conditional Routing
* Human-in-the-Loop Approval
* Email Tool Integration (Mock)

Tech Stack

* Python
* Gemini API
* LangGraph
* PyPDF
* dotenv
